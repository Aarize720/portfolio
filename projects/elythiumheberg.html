Hébergement Minecraft, web & bots Discord

Description

ElythiumHeberg est une plateforme d’hébergement dédiée aux serveurs Minecraft, sites web et bots Discord. Elle inclut un panel Pterodactyl personnalisé, un espace client intégré et des systèmes d’automatisation.

Technologies

  • Node.js · Backend API
  • Pterodactyl API · Gestion serveurs
  • MySQL · Stockage
  • CSS responsive · UI moderne
